Projects

NASA Multi-Modal Image Engineering Project

This project was developed as part of the Summer 2020 NASA OSTEM internship program. It involves the engineering of a pipeline to combine Geostationary Lightning Monitoring (GLM) data with Visible and Infrared imagery to improve Above Anvil Cirrus Plume (AACP) detection.

The final presentation slides for this project can be found here.

Micro Purge and Trap Device for GC on Aqueous Solutions

This project was developed as part of the 2021 Virginia Microelectronics Consortium Summer Scholars program. The purpose of the project was to develop a micro purge and trap device to integrate with existing gas chromatography (GC) technology to allow for portable GC systems.

The final presentation slides for this project can be found here.

Lepton Identification Neural Network

A project for distinguishing between electrons and muons in a set of simulated particle collision data. The prioject explores a compbination of machine learning and event reconstruction techniques for event classification.
